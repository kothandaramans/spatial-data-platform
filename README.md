# spatial-data-platform

## Endpoints:
    1. points: http://127.0.0.1:8000/spatial/v1/points/
        GET /points/
        POST /points/
        GET /points/{id}/
        PATCH /points/{id}/
        DELETE /points/{id}/
        
    2. polygons: http://127.0.0.1:8000/spatial/v1/polygons/
        GET /polygons/
        POST /polygons/
        GET /polygons/{id}/
        PATCH /polygons/{id}/
        DELETE /polygons/{id}/
